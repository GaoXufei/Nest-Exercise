
## NestJs BLOG API

```
基于Nest.js开发的BLOG SERVER层
目前正在完善，预计VIEW层会使用Angular or React其中一种来实现前后分离
```

## 下载方式

```
$ git clone https://github.com/GaoXufei/Nest-Exercise.git
```

```bash
$ npm install
or
$ yarn
```

## 运行方式

```bash
# development
$ npm run start
or
$ yarn start


# watch mode
$ npm run start:dev
or
$ yarn start:dev

# production mode
$ npm run start:prod
or
$ yarn start:prod
```

## 测试方式

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
